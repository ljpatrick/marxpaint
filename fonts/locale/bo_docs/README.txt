Tibetan TrueType Font (TTF) for Marx Paint

Bill Kendrick
bill@newbreedsoftware.com
http://www.newbreedsoftware.com/marxpaint/

March 29, 2006 - March 29, 2006


This font is required to run Marx Paint in Tibetan.
(e.g., with the "--lang tibetan" option)

To install, run "make install" as the superuser ('root').
The font file will be placed in the /usr/share/marxpaint/fonts/ directory.
