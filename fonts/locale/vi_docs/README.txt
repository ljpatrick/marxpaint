Vietnamese TrueType Font (TTF) for Marx Paint

Bill Kendrick
bill@newbreedsoftware.com
http://www.newbreedsoftware.com/marxpaint/

April 15, 2004 - April 15, 2004


This font is required to run Marx Paint in Vietnamese.
(e.g., with the "--lang vietnamese" option)

To install, run "make install" as the superuser ('root').
The font file will be placed in the /usr/share/marxpaint/fonts/locale/ directory.
