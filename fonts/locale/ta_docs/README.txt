README.txt for "marxpaint-ttf-tamil"
Tamil TrueType Font (TTF) for Marx Paint

Bill Kendrick
bill@newbreedsoftware.com
http://www.marxpaint.org/

December 5, 2002 - April 24, 2007


This font is required to run Marx Paint in Tamil.
(e.g., with the "--lang tamil" option)

To install, run "make install" as the superuser ('root').
The font file will be placed in the /usr/share/marxpaint/fonts/ directory.
