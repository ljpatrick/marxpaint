Telugu TrueType Font (TTF) for Marx Paint

Bill Kendrick
bill@newbreedsoftware.com
http://www.marxpaint.org/

April 24, 2007 - April 24, 2007


This font is required to run Marx Paint in Telugu
(e.g., with the "--lang telugu" option)

To install, run "make install" as the superuser ('root').
The font file will be placed in the /usr/share/marxpaint/fonts/ directory.
