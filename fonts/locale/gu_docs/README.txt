Hindi TrueType Font (TTF) for Marx Paint

Bill Kendrick
bill@newbreedsoftware.com
http://www.marxpaint.org/

October 12, 2006 - October 12, 2006


This font is required to run Marx Paint in Gujarati.
(e.g., with the "--lang gujarati" option)

To install, run "make install" as the superuser ('root').
The font file will be placed in the /usr/share/marxpaint/fonts/locale/
directory.
